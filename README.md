# 小刘的碎碎念 - 在线日记本

[![Website](https://img.shields.io/badge/网站-在线访问-blue)](https://crysyallll-sys.github.io/xiaoliu.github.io/)

这是一个功能完整的私人日记网页。它使用纯原生技术开发，无需任何后端或数据库，打开浏览器即可使用。数据会安全地保存在你自己的浏览器中。

## ✨ 功能特性

-   **📝 日记管理**：记录带标题和正文的日记，支持保存和删除。
-   **🌦️ 天气助手**：通过高德地图 API 实时查询并显示任意城市的天气。
-   **💿 音乐陪伴**：内置一个拟物风格的黑胶唱片机，点击即可播放/暂停音乐。
-   **🎨 精美界面**：模拟便签本的3D样式，带有动态飘浮的云朵背景。
-   **💾 数据本地化**：所有日记数据都存储在浏览器的 localStorage 中，关闭页面也不会丢失。
